Welcome to EasyMaker3D

EasyMaker3D is a desktop application for creating 3D-printable models. It is aimed at users who may not have much experience with 3D modeling and should be easier to learn than other modeling programs. With it, you can create many different types of models and export them as STL files that can be sent to a 3D printer.

EasyMaker3D is VR-enabled, meaning that it can be used in VR, but also works in a standard mouse+keyboard configuration. It can even be used with VR controllers without the headset on. In that mode, you can use any combination of controllers, mouse, and keyboard.

EasyMaker3D is available for Windows, Mac, and Linux. The Mac version may or not work in VR due to the lack of official SteamVR support.

Note

EasyMaker3D is pretty intuitive once you get to know a few basics. You can try out the Tutorials or take a look at the User Guide first. There is also a compact guide to keyboard shortcuts in the Cheat Sheet.

Installing and Running EasyMaker3D

  • Windows or Linux:

    • Open the the GitHub release page in a browser.

    • Download the Zip file for your operating system.

    • Extract all files wherever you like; this will create a folder called EasyMaker3D.

    • Double-click on the EasyMaker3D executable in this folder to run the application.

  • Mac:

    • Open the the GitHub release page in a browser.

    • Download the DMG file.

    • Double-click or install the DMG file to run the EasyMaker3D executable.

VR Only

Instructions for setting up your VR device for use with EasyMaker3D are on this page.

Problems?

If for some reason EasyMaker3D crashes, it saves a crash session file in the same directory it is run in. This file will contain the session up to the point of the crash, so you can load it to get back to where you were.

Feel free to open a new issue to report a bug or request a feature. Please include the crash file in your report if applicable.
