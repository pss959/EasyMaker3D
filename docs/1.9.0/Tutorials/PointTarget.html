Using the Point TargetÔÉÅ

This tutorial covers the use of the Point Target as an aid for model placement.

The Point Target can be placed on the stage or on the surface or bounds of any model in the scene.

When visible, the Point Target is used for the position and orientation of any new model that is added. It also serves as a snapping aid when translating or rotating a model.

The ring on the Point Target is used for Radial Layout, covered in another tutorial.
