Interacting with Panels in VRÔÉÅ

This tutorial covers the use of VR controllers with 2D panels in EasyMaker3D.

When an application panel or tool panel is visible, EasyMaker3D enters panel touch mode. In this mode, touch affordances are added to the controller avatars and panels are positioned closer to the viewer. This allows you to interact with certain panel UI elements by touching them with the affordance.

Grip dragging can be used to move and resize panels, using the handles on the sides and corners.

When a text input field is activated in full VR mode (with the headset on), a virtual keyboard panel appears. You can use the touch affordances to activate keys on this keyboard. In hybrid mode (headset off), you can use the regular keyboard.
