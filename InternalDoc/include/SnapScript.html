Command Arguments Effect
[whitespace] Blank lines are ignored.
# ... Comments are ignored, including inline comments.
action name Applies the named action.
click x y Simulates a mouse click at the given point in normalized window coordinates.
drag start|continue|end x y Simulates a mouse drag at the given point in normalized window coordinates.
hand L|R type Defines a controller model for the given hand. *type* is one of {Oculus_Touch, Vive, or None}.
hover x y Simulates a mouse hover at the given point in normalized window coordinates.
handpos L|R posx posy posz laserx lasery laserz guidex guidey guidez Moves the controller for the given hand to the given position (relative to the default rest position) and rotates so that the laser pointer and hover guide align with the given vectors. The hover guide direction can be zero if the default is ok.
headset on|off Simulates VR headset on or off.
key key_string Simulates a keyboard key press and release for the given key_string, which is in the same format as for keyboard shortcuts,
load [file_name] If a file name is given, loads a session from it. Otherwise, starts a new session. Session files are relative to PublicDoc/snaps/sessions.
mod on|off Turns modified interaction mode on or off.
select [name, ...] Selects the named models. If no names are given, deselects all models.
settings file_name Loads settings from the named file. Settings files are relative to PublicDoc/snaps/settings.
snap x y w h file_name Saves a snapshot image of the area of size wXh with (x,y) at the lower left to the named file. All values are in normalized window coordinates. Images are saved in PublicDoc/docs/images.
snapobj object_name margin file_name Saves a snapshot image of the area around the object named object_name with the given margin (in scene units) around all sides. Images are saved in PublicDoc/docs/images.
stage scale rot_angle Scales and rotates the stage by the given values.
stop Stops processing any more instructions. This can be used for testing to easily stop after a given instruction.
touch on|off Turns panel touch mode on or off.
view dirx diry dirz Changes the view to look along the given direction.
