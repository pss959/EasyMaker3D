Command Arguments Effect
[whitespace] Blank lines are ignored.
# ... Comments are ignored, including inline comments.
caption x y seconds text Displays a caption at the given location (center) for the given duration in seconds. A semicolon in the text can be used to split the caption over multiple lines.
click Simulates a mouse click at the current cursor position.
cursor on|off Shows or hides the cursor at the current location.
drag dx dy seconds [L|M|R] Simulates a mouse drag from the current cursor position by (dx,dy) in normalized window coordinates over the given duration in seconds. An optional argument can specify the left, middle, or right button.
highlight left bottom width height seconds Shows a highlight rectangle for the given duration in seconds.
highlightobj object_name seconds [margin] Shows a highlight rectangle around the object named object_name with the optional margin (in normalized window coordinate units) around all sides for the given duration in seconds. Note that the name may contain slashes to disambiguate it along a node path.
key key_string Simulates a keyboard key press and release for the given key_string, which is in the same format as for keyboard shortcuts,
mod on|off Turns modified interaction mode on or off for subsequent click and drag instructions.
moveover object_name seconds Moves the cursor from its current position to be over the named object in the scene, taking the given duration in seconds to do so. Note that the name may contain slashes to disambiguate it along a node path.
moveto x y seconds Moves the cursor (without interaction) from the current position to (x,y) in normalized window coordinates over the given duration in seconds.
section tag title Marks the current frame of the video as a section with the given tag and title.
start Starts capturing video. Any instructions before this are not part of the resulting video.
stop Stops processing any more instructions. This can be used for testing to easily stop after a given instruction.
tooltips on|off Turns tooltips on or off for subsequent hovers.
wait seconds Does nothing for the given duration in seconds.
