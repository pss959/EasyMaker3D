Command Arguments Effect
[whitespace] Blank lines are ignored.
# ... Comments are ignored, including inline comments.
caption x y seconds text Displays a caption at the given location (center) for the given duration in seconds.
click Simulates a mouse click at the current cursor position.
cursor on|off Shows or hides the cursor at the current location.
drag dx dy seconds Simulates a mouse drag from the current cursor position by (dx,dy) in normalized window coordinates over the given duration in seconds.
mod on|off Turns modified interaction mode on or off for subsequent click and drag instructions.
moveover object_name seconds Moves the cursor from its current position to be over the named object in the scene, taking the given duration in seconds to do so. If more than one object has the given name, slash-separated ancestors can be included. For example, Room/RotationTool/ZAxisRotator
moveto x y seconds Moves the cursor (without interaction) from the current position to (x,y) in normalized window coordinates over the given duration in seconds.
stop Stops processing any more instructions. This can be used for testing to easily stop after a given instruction.
wait seconds Does nothing for the given duration in seconds.
